## 📖 Sobre el Proyecto

**Diseño Web SMR** es un repositorio que recopila diversas prácticas, maquetaciones y componentes creados durante el módulo de Diseño de Interfaces Web. Es un showcase de mis habilidades progresivas en Frontend.

## ✨ Lo que encontrarás aquí

- 🏗️ **Maquetación Avanzada**: Uso de Flexbox y CSS Grid.
- 🎨 **Animaciones CSS**: Transiciones y keyframes.
- 📱 **Responsive Design**: Interfaces adaptables a cualquier resolución.
- 🧩 **Componentes UI**: Botones, tarjetas, modales, etc.
